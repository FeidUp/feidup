## 🎨 Design Patterns Used

### Color Usage
- **Primary Red-Orange**: CTAs, headings accents, icons
- **Cream/Off-White**: Backgrounds, gradient overlays
- **White**: Card backgrounds, main content areas
- **Gray Scale**: Body text, borders, subtle elements

### Component Patterns
```
Card Pattern:
└─ Gradient background (cream → white)
   └─ Rounded corners (3xl)
      └─ Shadow (lg/xl)
         └─ Icon (gradient circle)
            └─ Heading (Fredoka)
               └─ Body text (Inter)

CTA Pattern:
└─ Gradient button (red-orange)
   └─ Rounded full
      └─ Shadow (lg)
         └─ Hover: scale + shadow increase
```

### Animation Patterns
- **Fade-in**: Hero content (opacity + translateY)
- **Slide-in-right**: Hero images (opacity + translateX)
- **Hover scale**: Buttons (transform: scale(1.05))
- **Hover shadow**: Cards (shadow-lg → shadow-xl)

## 📱 Responsive Breakpoints

- **Mobile**: < 768px (1 column, hamburger menu)
- **Tablet**: 768px - 1024px (2 columns)
- **Desktop**: > 1024px (3 columns, full nav)
